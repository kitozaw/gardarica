31. August 2016

Конференция по ПОД/ФТ

Сегодня прошла очередная Конференция по ПОД/ФТ для специалистов КПК. Степин Алексей Викторович  - независимый эксперт в области противодействия легализации (отмывания) доходов, рассказал собравшимся об изменениях которые нужно учитывать в своей работе.


О проекте:

В связи с постоянно усиливающимся документооборотом для КПК, в частности перед контролирующими органами, Центр Обслуживания кредитных союзов (ЦОКС) г. Санкт -Петербург разработал для членов Ассоциации "Гардарика" услугу "Консалтинг - Проект по 115 ФЗ".  У нас в группе есть свободные места. Кого заинтересует данное предложение, пожалуйста свяжитесь с нами.

Что предлагаем:

В рамках информационно - консультационного обслуживания Ваш кооператив получит:
-  Локально – нормативные документы с января 2016 года для Вашего КПК:

 

  • Правила внутреннего контроля ПОД/ФТ, учитывая специфику деятельности Вашего КПК;
  • Первичную документацию по ПОД/ФТ  для Вашего КПК;
  • Актуализацию Правил внутреннего контроля ПОД/ФТ по мере изменений законодательства и на основе складывающейся практики проверок контролирующих органов в каждом федеральном округе и пр.

 

- А так же 1 раз в месяц мы будем информировать специалистов Вашего КПК об изменениях законодательных актов ПОД/ФТ в форме Скайп-конференции.

Дата и время такой конференции будет согласовываться дополнительно.

 

 
О консультанте:
Для осуществления обслуживания КПК по информационно - консультативному соровождению мы пригласили опытного специалиста.

Степин Алексей Викторович  - независимый эксперт в области противодействия легализации (отмывания) доходов, в прошлом – сотрудник Росфинмониторинга. Обладает обширным опытом проведения целевых инструктажей ПОД/ФТ (за 3,5 года проведено более 80 инструктажей, несколько тысяч слушателей), имеет Свидетельство МУМЦФМ о прохождении обучения по программе «Подготовка кадров в сфере противодействия отмыванию преступных доходов, и финансированию терроризма».
 
Стоимость услуг:
Для члена организации - партнера Ассоциации "Гардарики" - 5000 р. в месяц;
Для кооперативов Ассоциации "Гардарика" - 3500 р. в месяц;
Для членов МРК - 4000 р. в месяц;
Для первого зарегистрировавшегося - 4000 р. в месяц
